[instrumentr](https://prl-prg.github.io/instrumentr/) is a lightweight framework for dynamic analysis of R.


# Administration

To install:

```sh
$ R CMD INSTALL instrumentr
```

To uninstall:

```sh
$ R -e "remove.packages('instrumentr')"
```
